## Getting Started
1. clone the repo
```
git clone https://github.com/ckdtjs505/notflix
```

2. Install NPM packages
```
npm install
```

3. start dev server
```
npm run dev:server
```

4. start dev:webpack

```
npm run dev:assets
```
